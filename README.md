# 🎬 Nozes Audiovisual | Landing Page

Uma Landing Page completa e responsiva desenvolvida do zero para ser a vitrine digital da minha produtora, a **Nozes Audiovisual**.

Este projeto marca um momento importante na minha jornada de unir o empreendedorismo prático (liderando os projetos da produtora) com o meu desenvolvimento técnico como estudante de tecnologia. 

🔗 **Acesse o site rodando aqui:** [Nozes Audiovisual Criativo](https://nozesaudiovisualcriativo.netlify.app/)

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído "na unha", sem uso de templates ou frameworks pesados, focando em performance e fundamentos da web:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout em Grid/Flexbox e responsividade para dispositivos móveis.
* **JavaScript (Vanilla):** Lógica para o menu interativo, carrossel de imagens e o filtro dinâmico da galeria de vídeos.

## ✨ Funcionalidades e Destaques

Apesar de ser um projeto inicial, implementei recursos essenciais para uma página profissional de alta conversão:

* **Portfólio com Filtro Dinâmico:** Os usuários podem filtrar os vídeos por categoria (Casamentos ou Institucional & Drone) sem recarregar a página.
* **Carrossel Automático:** Uma galeria de imagens de bastidores que roda automaticamente.
* **Design Responsivo:** O layout se adapta perfeitamente a celulares, tablets e desktops.
* **SEO e Open Graph:** Meta tags configuradas para ranqueamento no Google e para gerar um "cartão de visitas" digital perfeito ao compartilhar o link no WhatsApp e redes sociais.
* **Google Analytics 4 (GA4):** Rastreamento de acessos em tempo real e eventos customizados para monitorar cliques nos botões de conversão (WhatsApp).
